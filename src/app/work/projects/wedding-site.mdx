## Resumen

Landing page desarrollada para gestionar la asistencia de invitados a un casamiento, ofreciendo una experiencia simple, estética y funcional tanto en desktop como en mobile. Implementado con **Next.js**, **TypeScript**, **Once UI**, **Sass** y una integración personalizada con **Google Sheets** como back-end sin servidor.

## Objetivo

Crear un sitio liviano y elegante que permitiera a los invitados confirmar su asistencia de manera simple, evitando formularios complejos o necesidad de autenticación.

## Tecnologías utilizadas

- **Next.js** – Framework React para rendering híbrido.
- **TypeScript** – Tipado estático para mayor robustez.
- **Once UI** – Sistema de diseño reutilizable para consistencia visual.
- **Sass** – Estilado modular y escalable.
- **Google Sheets** – Actúa como base de datos a través de una API personalizada.

## Funcionalidades principales

- **Formulario personalizado**: Captura nombre, apellido, cantidad de acompañantes y necesidades especiales de alimentos.
- **Campos condicionales**: Se muestran según el número de asistentes.
- **Validación y feedback**: Alertas visuales para errores o éxito.
- **Integración con Google Sheets**: Registros almacenados automáticamente vía API.
- **Diseño responsive**: 100% adaptado a mobile.
